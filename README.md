Nome do projeto: Sistema Skillmatch - Compatibilidade entre vagas e candidatos.

O sistema Skillmatch resolve o problema ao qual proposto foi proposto, ser uma aplicação web (Single Page) em HTML, CSS e JavaScript puro, reaproveitando o motor de compatibilidade que foi desenvolvido a partir do projeto anterior.

A maioria das tecnologias utilizadas se encontram tanto na documentação do desafio em si quanto na própria aplicação, mas aqui vão algumas das principais tecnologias utilizadas:
*   HTML: section, form, nav, a, fieldset, h1, h2, h3, img, etc;
*   Atributos HTML: for, id, aria-live, name, value, class, src, href, etc;
*   CSS: *, .classe, color, diplay: flex, gap, font-family, margin, etc;
*   JavaScript: métodos de array, operadores, operadores ternários, if-else, import, export, classe, métodos de classe, dentre outra mais.

Para executar o projeto basta apenas abrir a extensão do program Visual Studio Code no index.html da pasta do projeto que o resto da aplicação será carregada automaticamente, caso não funcione também é possível acessar a aplicação pelo link a seguir:
-   https://kauan-kantovick.github.io/Skillmatch-Web-SCTEC/

Existem muitas funcionalidades interessantes ao qual seriam muito interessantes que a aplicação possuísse, creio que além de todos os requisitos funcionais e adicionais que o edital do projeto possui creio que:
*   O projeto possui um integração com uma API ou banco de dados com diversos usuários e vagas cadastras, dando um ar de uma aplicaçâo real e com centenas de usuários;
*   A aplicação possuir um filtro por área profissional, onde aparecesse apenas as vagas determinada área;
*   Um sistema de feedback onde os usuários poderiam sugerir melhorias ou alertar sobre erros na aplicação.
